# Writing samples

Put 4-8 of your own writing samples here (in a subfolder named for your voice profile,
e.g. `sai/`), then derive a Style Bible in `voice/profiles/<name>.md` from them.

Samples are personal, so this directory is gitignored by default (see the skill's
`.gitignore`): your raw writing is not committed or published. The distilled Style Bible
in `voice/profiles/` is what the pipeline actually reads, and it is the only voice artifact
that needs to be shared.
